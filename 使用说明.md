# 个人网站使用说明

## 快速开始

本项目改为“线上部署 + 服务端代理调用智谱”，不再使用本地服务。

### 免费部署（推荐：Vercel）
1. 注册/登录 Vercel。
2. 新建项目并导入当前代码仓库（或上传此文件夹）。
3. 在 Vercel 的项目设置中新增环境变量：
   - `ZHIPU_API_KEY` = 你的智谱 API Key
4. 部署完成后，访问 Vercel 提供的线上地址。

> API Key 仅存储在服务端环境变量中，前端不会展示。

## 设计说明

本网站采用**简约高级风**设计：

- **色彩**：莫兰迪色系（浅灰蓝 #E0E9F0、暖米白 #F9F7F4、深灰 #333333、暖橙 #F0B86E）
- **布局**：顶部导航 → 英雄区（含照片轮播）→ 过渡区 → 底部社交媒体
- **交互**：轮播 3 秒自动切换、悬停暂停、移动端可滑动切换
- **适配**：响应式设计，兼容电脑、平板、手机

## 自定义配置

### 1. 个人信息
- **Logo/名字**：搜索 `XXX's Portfolio` 替换为您的名字
- **主标题**：修改 `你好，我是XXX`
- **副标题**：修改 `生活记录者 / 设计师 / 摄影师`
- **版权信息**：修改底部 `© 2026 XXX 保留所有权利`

### 2. 照片轮播
- 在 `index.html` 中搜索 `carousel__slide`，找到轮播图片区域
- 将 `src` 属性替换为您的图片 URL
- 建议图片尺寸：1920×1080（16:9），已做压缩处理以保证加载速度
- 可增删 `<div class="carousel__slide">` 块以调整照片数量

### 3. 社交媒体链接
在底部 Footer 区域，修改各图标的 `href` 属性：
- 小红书：`https://www.xiaohongshu.com/user/profile/xxx`
- 微博：`https://weibo.com/xxx`
- 微信：可保留 `onclick` 弹窗提示，或改为您的微信二维码页面
- Instagram：`https://www.instagram.com/xxx`
- B站：`https://space.bilibili.com/xxx`

### 4. 过渡区文案
修改 `用镜头记录生活，用热爱奔赴山海` 为您的个性化文字。

## 图片本地使用

若使用本地图片，请将图片放在与 `index.html` 同目录或子目录下，例如：

```html
<img src="images/photo1.jpg" alt="描述">
```

并确保 `images` 文件夹中有对应图片文件。

## 浏览器兼容

支持 Chrome、Firefox、Safari、Edge 等现代浏览器，以及手机和平板设备。
